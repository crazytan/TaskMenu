TaskMenu

A lightweight, native macOS menu bar app for Google Tasks.

Features

  • Menu bar native Lives in the menu bar with no Dock icon or main window. Always one click away.
  • Google Tasks sync Sign in with Google OAuth 2.0. Create, edit, complete, and delete tasks.
  • Quick add Capture tasks inline without leaving what you're doing.
  • Multiple lists Switch between your Google Task lists with a single click.
  • Due dates Set and view due dates on your tasks.
  • Secure OAuth tokens stored in the macOS Keychain. No data leaves your device.
  • Zero dependencies Built with Swift 6 and SwiftUI. Apple frameworks only.
  • Open source Free and open source under the GNU GPLv3 license.

Get Started

  1. Clone the repo: git clone https://github.com/crazytan/TaskMenu.git
  2. Set up Google OAuth credentials in Config.xcconfig
  3. Generate the project: xcodegen generate
  4. Open in Xcode and build

Requires macOS 14+, Xcode 16+, and XcodeGen.
